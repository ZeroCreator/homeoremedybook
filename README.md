# Homeo Remedy Book

Приложение для справочного материала по гомеопатии

Установить все зависимости:

```bash
$ pip install -r requirements.txt
```

Запустите приложение:

```bash
$ python wsgi.py
```

Откройте в браузере:
http://localhost:5000
